> 가벼운 계획이 아니라, 스스로와 약속을 지키기 위해 기록하는 것

- 하루에 오직 세 가지 목표만 세울 수 있는 단순하지만 강력한 `나 자신과의 약속 플래너`입니다.
- 한 번 세운 목표는 **당일 수정이 불가능**하고, 하루가 끝나면 달성 여부를 기록하게 됩니다.
- 그리고 친구들과 서로의 달성 현황을 피드에서 확인하며 완수해야겠다는 마음을 자극합니다.
